JavaScript is Disabled!

Please enable JavaScript to fully experience the site. Please follow the steps below to enable JavaScript on your browser.

If you're unsure how to enable JavaScript, visit the official help pages linked below for detailed instructions.

Google Chrome (Desktop & Mobile)

  • Open Chrome and go to Settings .
  • Click on Privacy and security .
  • Go to Site Settings .
  • Find JavaScript and turn on Allowed (recommended).
  • For more help, visit Google Support .

Mozilla Firefox

  • Type about:config in the address bar and press Enter.
  • Click Accept the Risk and Continue .
  • Search for javascript.enabled and toggle it to true.
  • For more help, visit Mozilla Support .

Microsoft Edge

  • Open Edge and go to Settings .
  • Navigate to Cookies and site permissions .
  • Find JavaScript and turn it on.
  • For more help, visit Microsoft Support .

Opera

  • Go to Settings .
  • Click on Site Settings .
  • Find JavaScript and enable it.
  • For more help, visit Opera Help .

Safari (Mac & iOS)

  • Go to Safari > Preferences .
  • Select the Security tab.
  • Check Enable JavaScript .
  • For iPhone/iPad: Go to Settings >Safari >Advanced and enable JavaScript.
  • For more help, visit Apple Support .

Internet Explorer

  • Go to Tools > Internet Options .
  • Select Security >Custom Level.
  • Find Active Scripting and enable it.
  • For more help, visit Microsoft Support .
